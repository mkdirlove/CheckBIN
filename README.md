#### Installation

Copy-paste this into your terminal:

```
git clone https://github.com/mkdirlove/CheckBIN.git
```
```
cd CheckBIN
```
```
python3 CheckBIN.py -h
```
#### Usage
```
____ _  _ ____ ____ _  _ ___  _ _  _ 
|    |__| |___ |    |_/  |__] | |\ | 
|___ |  | |___ |___ | \_ |__] | | \|

usage: CheckBIN.py [-h] --bin BIN

CheckBIN - BIN Checker tool

optional arguments:
```
